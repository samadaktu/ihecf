# IHECF - Indian Higher Education & Career Fairs

This is the professional website for IHECF, connecting students with top Indian universities worldwide.

## Project Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- Modern React + Vite architecture
- Tailwind CSS for styling
- Responsive design for all devices
- Dynamic routing for services and countries
- Interactive image gallery and contact forms
